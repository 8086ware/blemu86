Blemu86 is an IBM PC XT compatible emulator written in C++. You need a bios rom to run it, currently it only accepts bios changes inside of the main.cpp file (inside the initialization string argument). I recommend https://github.com/640-KB/GLaBIOS and you can change the floppies in 86blemu.h. Obviously a work in progress, because of those 2 previous things...
